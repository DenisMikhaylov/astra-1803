### Лабораторная работа №4: «Условные операторы и проверки в Bash»

**Цель работы:** Научиться использовать условный оператор `if` для ветвления логики скрипта, применять различные проверки (файлов, строк, чисел) и комбинировать их с переменными.

**Теоретическая справка:**

В Bash условия проверяются с помощью команды `[` (синоним `test`). Основные типы проверок:

**Файловые проверки:**
- `-f` — существует ли обычный файл
- `-d` — существует ли директория
- `-e` — существует ли объект (файл/директория)
- `-r` — доступен для чтения
- `-w` — доступен для записи
- `-x` — доступен для выполнения
- `-s` — файл не пустой

**Строковые проверки:**
- `-z` — строка пуста (zero length)
- `-n` — строка не пуста (non-zero)
- `=` или `==` — строки равны
- `!=` — строки не равны
- `<` и `>` — лексикографическое сравнение (в `[[ ]]`)

**Числовые проверки:**
- `-eq` — равно (equal)
- `-ne` — не равно (not equal)
- `-lt` — меньше (less than)
- `-le` — меньше или равно
- `-gt` — больше (greater than)
- `-ge` — больше или равно

**Логические операторы:**
- `&&` — И (AND) — в `[ ]` использовать `-a`, в `[[ ]]` — `&&`
- `||` — ИЛИ (OR) — в `[ ]` использовать `-o`, в `[[ ]]` — `||`
- `!` — НЕ (NOT)

**Рекомендация:** Используйте двойные квадратные скобки `[[ ]]` — они мощнее и безопаснее.

### Подсказки и рекомендации

1. **Для числовых сравнений всегда используйте `-eq`, `-lt`, `-gt`**, а не `=`!
2. **Проверка на число:**

   ```bash
   fi
   ```
3. **Проверка на пустую строку:** `if [[ -z "$VAR" ]]`
4. **Комбинирование условий:**
   ```bash
   if [[ -f "$FILE" && -r "$FILE" ]]; then
       echo "Файл существует и доступен для чтения"
   fi
   ```
5. **Проверка наличия команды:**
   ```bash
   if command -v bc &>/dev/null; then
       echo "bc установлен"
   fi
   ```

### Расшифровка конструкций

- `[[ condition ]]` — расширенный синтаксис проверки (рекомендую).
- `[ condition ]` — классический синтаксис (совместимый с sh).
- `[[ -z "$VAR" ]]` — истина, если переменная пуста.
- `[[ -n "$VAR" ]]` — истина, если переменная не пуста.
- `[[ "$A" == "$B" ]]` — строки равны.
- `[[ "$A" < "$B" ]]` — лексикографически меньше (внимание с перенаправлениями!).

```bash
#!/usr/bin/env bash

# Лабораторная работа №4, Вариант 1
# Тема: Условные операторы, проверка файлов и директорий

# Проверяем, передан ли аргумент
if [[ -z "$1" ]]; then
    echo "Ошибка: не указан путь к файлу или директории"
    echo "Использование: $0 <путь>"
    exit 1
fi

PATH_TO_CHECK="$1"

# Проверяем, существует ли объект
if [[ ! -e "$PATH_TO_CHECK" ]]; then
    echo "Ошибка: $PATH_TO_CHECK не существует!"
    exit 1
fi

# Объект существует
echo "Объект $PATH_TO_CHECK существует."

# Определяем тип
if [[ -f "$PATH_TO_CHECK" ]]; then
    echo "Тип: обычный файл"
    
    # Размер файла
    SIZE=$(stat -c%s "$PATH_TO_CHECK" 2>/dev/null || stat -f%z "$PATH_TO_CHECK" 2>/dev/null)
    echo "Размер: $SIZE байт"
    
    # Проверка на пустоту
    if [[ -s "$PATH_TO_CHECK" ]]; then
        echo "Пустой: нет"
    else
        echo "Пустой: да"
    fi
    
elif [[ -d "$PATH_TO_CHECK" ]]; then
    echo "Тип: директория"
    
    # Количество элементов в директории
    COUNT=$(ls -1 "$PATH_TO_CHECK" 2>/dev/null | wc -l)
    echo "Количество элементов: $COUNT"
else
    echo "Тип: другой (специальный файл, сокет и т.д.)"
fi

# Проверка прав доступа
echo "Права:"
if [[ -r "$PATH_TO_CHECK" ]]; then
    echo "  чтение (+) "
else
    echo "  чтение (-) "
fi

if [[ -w "$PATH_TO_CHECK" ]]; then
    echo "  запись (+) "
else
    echo "  запись (-) "
fi

if [[ -x "$PATH_TO_CHECK" ]]; then
    echo "  выполнение (+) "
else
    echo "  выполнение (-) "
fi

exit 0
```

```bash
#!/usr/bin/env bash

# Лабораторная работа №4, Вариант 2
# Тема: Калькулятор с проверками

# Проверка количества аргументов
if [[ $# -ne 3 ]]; then
    echo "Ошибка: требуется 3 аргумента"
    echo "Использование: $0 <число1> <оператор> <число2>"
    echo "Операторы: +, -, *, /"
    exit 1
fi

NUM1="$1"
OPERATOR="$2"
NUM2="$3"

# Проверка, что аргументы 1 и 3 являются числами
if [[ ! "$NUM1" =~ ^-?[0-9]+$ ]]; then
    echo "Ошибка: '$NUM1' не является числом!"
    exit 1
fi

if [[ ! "$NUM2" =~ ^-?[0-9]+$ ]]; then
    echo "Ошибка: '$NUM2' не является числом!"
    exit 1
fi

# Выполнение операции
case "$OPERATOR" in
    +)
        RESULT=$((NUM1 + NUM2))
        ;;
    -)
        RESULT=$((NUM1 - NUM2))
        ;;
    '*')
        RESULT=$((NUM1 * NUM2))
        ;;
    /)
        # Проверка деления на ноль
        if [[ $NUM2 -eq 0 ]]; then
            echo "Ошибка: деление на ноль!"
            exit 1
        fi
        RESULT=$((NUM1 / NUM2))
        ;;
    *)
        echo "Ошибка: неподдерживаемый оператор '$OPERATOR'"
        echo "Используйте: +, -, *, /"
        exit 1
        ;;
esac

echo "$NUM1 $OPERATOR $NUM2 = $RESULT"
exit 0
```

#### Вариант 3 (Мониторинг системы)

```bash
#!/usr/bin/env bash

# Лабораторная работа №4, Вариант 3
# Тема: Мониторинг системы

# Установка порога
if [[ -z "$1" ]]; then
    THRESHOLD=80
    echo "Порог не указан, установлен по умолчанию: $THRESHOLD%"
else
    # Проверка, что порог - число
    if [[ ! "$1" =~ ^[0-9]+$ ]]; then
        echo "Ошибка: порог должен быть числом!"
        exit 1
    fi
    
    THRESHOLD="$1"
    
    # Проверка диапазона
    if [[ $THRESHOLD -lt 0 || $THRESHOLD -gt 100 ]]; then
        echo "Ошибка: порог должен быть в диапазоне 0-100!"
        exit 1
    fi
fi

# Получение загрузки CPU
# Используем top в пакетном режиме
LOAD=$(top -bn1 | grep "Cpu(s)" | awk '{print $2}' | cut -d'%' -f1)

# Если не получилось через top, пробуем через uptime
if [[ -z "$LOAD" ]]; then
    LOAD=$(uptime | awk -F'load average:' '{print $2}' | cut -d, -f1 | sed 's/ //g')
    # Преобразуем в проценты (приблизительно)
    LOAD=$(echo "$LOAD * 100" | bc 2>/dev/null | cut -d. -f1)
fi

# Проверка, что загрузка получена
if [[ -z "$LOAD" ]]; then
    echo "Ошибка: не удалось получить загрузку CPU!"
    exit 1
fi

echo "Текущая загрузка CPU: $LOAD%"

# Сравнение с порогом
if [[ $LOAD -lt $THRESHOLD ]]; then
    echo "Система работает стабильно (порог: $THRESHOLD%)"
elif [[ $LOAD -ge $THRESHOLD && $LOAD -lt $((THRESHOLD + 10)) ]]; then
    echo "ВНИМАНИЕ: Загрузка выше нормы!"
else
    echo "КРИТИЧЕСКАЯ ЗАГРУЗКА!"
fi

exit 0
```

#### Вариант 4 (Управление пользователями)

```bash
#!/usr/bin/env bash

# Лабораторная работа №4, Вариант 4
# Тема: Управление пользователями

# Проверка прав root
if [[ $EUID -ne 0 ]]; then
    echo "ВНИМАНИЕ: Некоторые операции требуют прав root"
    echo "Используйте: sudo $0"
    echo ""
fi

# Проверка количества аргументов
if [[ $# -lt 2 ]]; then
    echo "Использование: $0 <команда> <имя_пользователя>"
    echo "Команды: check, create, delete"
    exit 1
fi

COMMAND="$1"
USERNAME="$2"

# Проверка существования пользователя
if id "$USERNAME" &>/dev/null; then
    USER_EXISTS=1
else
    USER_EXISTS=0
fi

case "$COMMAND" in
    check)
        if [[ $USER_EXISTS -eq 0 ]]; then
            echo "Пользователь $USERNAME не найден."
            exit 1
        fi
        
        echo "Пользователь $USERNAME найден:"
        echo "  UID: $(id -u "$USERNAME")"
        echo "  GID: $(id -g "$USERNAME")"
        echo "  Домашняя директория: $(eval echo ~"$USERNAME")"
        echo "  Оболочка: $(getent passwd "$USERNAME" | cut -d: -f7)"
        ;;
        
    create)
        if [[ $USER_EXISTS -eq 1 ]]; then
            echo "Ошибка: пользователь $USERNAME уже существует!"
            exit 1
        fi
        
        if [[ $EUID -ne 0 ]]; then
            echo "Ошибка: для создания пользователя требуются права root!"
            exit 1
        fi
        
        useradd -m -s /bin/bash "$USERNAME"
        if [[ $? -eq 0 ]]; then
            echo "Пользователь $USERNAME успешно создан."
        else
            echo "Ошибка: не удалось создать пользователя!"
            exit 1
        fi
        ;;
        
    delete)
        if [[ $USER_EXISTS -eq 0 ]]; then
            echo "Ошибка: пользователь $USERNAME не существует!"
            exit 1
        fi
        
        if [[ $EUID -ne 0 ]]; then
            echo "Ошибка: для удаления пользователя требуются права root!"
            exit 1
        fi
        
        read -p "Вы уверены, что хотите удалить пользователя $USERNAME? (y/n): " CONFIRM
        
        if [[ "$CONFIRM" == "y" || "$CONFIRM" == "Y" ]]; then
            userdel -r "$USERNAME"
            if [[ $? -eq 0 ]]; then
                echo "Пользователь $USERNAME удален."
            else
                echo "Ошибка: не удалось удалить пользователя!"
                exit 1
            fi
        else
            echo "Операция отменена."
        fi
        ;;
        
    *)
        echo "Ошибка: неизвестная команда '$COMMAND'"
        echo "Доступные команды: check, create, delete"
        exit 1
        ;;
esac

exit 0
```

#### Вариант 5 (Работа с датами и бэкапами)

```bash
#!/usr/bin/env bash

# Лабораторная работа №4, Вариант 5
# Тема: Управление бэкапами

# Проверка аргументов
if [[ -z "$1" ]]; then
    echo "Ошибка: не указана директория для бэкапа"
    echo "Использование: $0 <директория> [дни_хранения]"
    exit 1
fi

SOURCE_DIR="$1"
RETENTION_DAYS="${2:-7}"

# Проверка, что второй аргумент - число
if [[ ! "$RETENTION_DAYS" =~ ^[0-9]+$ ]]; then
    echo "Ошибка: '$RETENTION_DAYS' не является числом!"
    echo "Установлено значение по умолчанию: 7 дней"
    RETENTION_DAYS=7
fi

# Проверка существования исходной директории
if [[ ! -d "$SOURCE_DIR" ]]; then
    echo "Ошибка: директория $SOURCE_DIR не существует!"
    exit 1
fi

# Создание директории для бэкапов
BACKUP_DIR="${HOME}/backups"
if [[ ! -d "$BACKUP_DIR" ]]; then
    mkdir -p "$BACKUP_DIR"
    if [[ $? -ne 0 ]]; then
        echo "Ошибка: не удалось создать директорию $BACKUP_DIR"
        exit 1
    fi
    echo "Создана директория для бэкапов: $BACKUP_DIR"
fi

# Создание бэкапа
TIMESTAMP=$(date "+%Y%m%d_%H%M%S")
BACKUP_NAME="backup_${TIMESTAMP}.tar.gz"
BACKUP_PATH="${BACKUP_DIR}/${BACKUP_NAME}"

echo "Создание бэкапа: $BACKUP_NAME"

tar -czf "$BACKUP_PATH" -C "$(dirname "$SOURCE_DIR")" "$(basename "$SOURCE_DIR")" 2>/dev/null

if [[ -f "$BACKUP_PATH" ]]; then
    SIZE=$(du -h "$BACKUP_PATH" | cut -f1)
    echo "Размер: $SIZE"
    echo "Бэкап успешно создан"
else
    echo "Ошибка: не удалось создать бэкап!"
    exit 1
fi

# Удаление старых бэкапов
echo "Поиск бэкапов старше $RETENTION_DAYS дней..."

OLD_BACKUPS=$(find "$BACKUP_DIR" -name "backup_*.tar.gz" -type f -mtime +"$RETENTION_DAYS" 2>/dev/null)

if [[ -n "$OLD_BACKUPS" ]]; then
    COUNT=$(echo "$OLD_BACKUPS" | wc -l)
    echo "Найдено старых бэкапов: $COUNT"
    
    echo "$OLD_BACKUPS" | while read -r file; do
        rm -f "$file"
        if [[ $? -eq 0 ]]; then
            echo "  Удален: $(basename "$file")"
        fi
    done
    echo "Удалено старых бэкапов: $COUNT"
else
    echo "Старых бэкапов не найдено"
fi

# Вывод статистики
echo ""
echo "--- Статистика бэкапов ---"
TOTAL=$(find "$BACKUP_DIR" -name "backup_*.tar.gz" -type f 2>/dev/null | wc -l)
TOTAL_SIZE=$(du -sh "$BACKUP_DIR" 2>/dev/null | cut -f1)
echo "Всего бэкапов: $TOTAL"
echo "Общий размер: ${TOTAL_SIZE:-0}"

exit 0
```
