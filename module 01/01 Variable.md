
### Лабораторная работа №1: «Переменные и подстановки в Bash»

### Варианты заданий (Выберите один по списку группы)

#### Вариант 1 (Системный администратор)
Напишите скрипт, который:
1. Принимает на вход **путь к директории** и **расширение файла** (например, `.txt`).
2. Создает строковую переменную `RESULT`, в которую записывает команду `ls -l`.
3. Создает целочисленную переменную `COUNT`, равную количеству файлов с данным расширением в указанной директории (используйте подстановку `$(...)`).
4. Создает массив `FIRST_FIVE`, который содержит первые 5 имен файлов из вывода `ls`.
5. Создает переменную `REPORT`, которая содержит строку: *"В папке [путь] найдено [COUNT] файлов с расширением [расширение]. Первые два: [элемент0], [элемент1]"*.
6. Выводит `REPORT` на экран.

#### Вариант 2 (Аналитик данных)
Напишите скрипт, который:
1. Принимает строку с произвольным текстом через `read -p`.
2. Сохраняет её в переменную `INPUT_TEXT`.
3. Создает переменные:
   - `LEN` = длина текста.
   - `FIRST` = первый символ текста.
   - `LAST` = последний символ текста.
   - `CLEAN` = текст, преобразованный в нижний регистр.
4. Создает **индексированный массив** `WORDS`, который получается разбиением `INPUT_TEXT` по пробелам (используйте `()` и подстановку).
5. Создает строку `STATS`, которая содержит количество слов (`${#WORDS[@]}`), первый и последний символы.
6. Все вычисления выполняются **без использования внешних утилит** (например, `wc`, `tr`, `sed` не использовать, только встроенные средства Bash). Вывод `STATS`.

#### Вариант 3 (Сетевой инженер)
Напишите скрипт, который:
1. Принимает **IP-адрес** (например, `192.168.1.1`) и **порт** (например, `8080`) как аргументы командной строки (`$1`, `$2`).
2. Создает переменную `PROTOCOL` со значением `"http"`.
3. Создает целочисленную переменную `PORT_CHECK`, в которую записывается порт, но если порт не передан (`$2` пустой), то присвоить `80`.
4. Создает массив `URL_PARTS`, состоящий из: `PROTOCOL`, `$1`, `PORT_CHECK`.
5. Создает строковую переменную `FULL_URL` = `"$PROTOCOL://$1:$PORT_CHECK"`.
6. Создает переменную `PING_CMD` = `"ping -c 1 "` + первый октет IP-адреса (извлеките через подстановку `${var%%.*}` или другие манипуляции).
7. Выводит `FULL_URL` и `PING_CMD`.
