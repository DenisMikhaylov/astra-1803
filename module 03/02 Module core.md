Лабораторная работа  «Управление модулями ядра в Linux»
Цель работы: Научиться управлять модулями ядра, загружать, выгружать и настраивать параметры модуля e1000e для сетевых карт Intel.

Теоретическая справка:

Что такое модуль ядра?

Модуль ядра — это фрагмент кода, который может загружаться и выгружаться в ядро во время работы системы.

Модули позволяют добавлять поддержку оборудования, файловых систем и других функций без перекомпиляции ядра.

Модуль e1000e:

Драйвер для сетевых карт Intel серий PRO/1000

Поддерживает чипсеты: 82571, 82572, 82573, 82574, 82583, I217, I218, I219

Широко используется в серверах и рабочих станциях

Основные команды для работы с модулями:

```
# Просмотр загруженных модулей
lsmod

# Информация о модуле
modinfo e1000e

# Загрузка модуля
modprobe e1000e

# Выгрузка модуля
rmmod e1000e

# Настройка параметров модуля
modprobe e1000e param=value

# Просмотр параметров модуля
cat /sys/module/e1000e/parameters/param

```
